=== 1. Isolation credentials Docker Hub

* **Fichier** : `dockerhub-creds.yml` (non tracké, ajouté à `.gitignore`)

* **Sécurité** : `.gitignore` mis à jour pour ne jamais tracker `dockerhub-creds.yml`

=== 2. Script de push local

* **Fichier** : `scripts/dockerhub-push.sh`
* **Fonctionnement** :
  - Parse `dockerhub-creds.yml` avec `grep` + `sed`
  - Authentification via `docker login -u user --password-stdin` (token jamais visible en CLI)
  - Build + push avec `docker buildx build --push`
  - Tags : `cheroliv/magic-stick-cli:${VERSION}` + `:latest`

=== 3. Tasks Gradle Docker

Ajout dans `build.gradle.kts` :

[source,kotlin]
----
tasks.register<Exec>("dockerHubLogin")   // login local uniquement
tasks.register<Exec>("dockerBuildCli") // build image CLI
tasks.register<Exec>("dockerPushCli")  // build + push (depend de dockerHubLogin)
----

=== 4. Test push local — REUSSI
