=== Bugs corriges

==== squashfs > 4 Go dans ISO
**Cause** : xorriso par defaut refuse un fichier isole > 4 Go (limitation ISO 9660 Level 1)
**Fix** : `-iso-level 3` dans la commande xorriso via patch python dans `build-inner.sh`
**Reference** : xorriso(1) - `-iso-level 3` permet fichiers > 4 Go

==== test-software.sh et loop devices CI/CD
**Cause** : `mount -o loop ... squashfs` echoue dans Docker non-privileged sur les runners GitHub
**Fix** : Remplacer par `unsquashfs -q -d ...` pour extraire le squashfs sans loop device

==== Path squashfs live/ vs casper/
**Cause** : Xubuntu 24.04 utilise `casper/filesystem.squashfs`, pas `live/filesystem.squashfs`
**Fix** : Detection dynamique : si `live/` n'existe pas â†’ fallback sur `casper/`

==== NVM dans le chroot
**Cause** : `curl ... | sh` est interactif et depend de `$HOME/.bashrc` pour creer `nvm.sh`
**Fix** : Clone direct du repo `nvm` via git sur la branche `v0.40.4`
