== Lecons apprises

* SourceForge n'a pas de limite de taille — xz n'est necessaire QUE pour GitHub Releases (2 Go max)
* Une ISO squashfs est deja compressee en interne : xz par-dessus ajoute peu de gain

* L'erreur xz "Cannot set the file group" est un bug de permission dans GitHub Actions runners — avoidable en supprimant xz
* Le footer du site de doc doit pointer vers le site du developpeur pour la navigabilite cross-domaine
