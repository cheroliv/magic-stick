= EPIC V-1 — Validation Physique ISO
:author: cheroliv
:date: 2026-05-22
:status: EN COURS
:toc: left

== Objectif

Tester le boot de l'ISO magic-stick **sur du matériel réel** (PC physique).
Ce que QEMU ne peut pas valider : pilotes GPU réels, Wi-Fi/Bluetooth, chipset,
USB, performance réelle.

== ISO à tester

* Version : **0.1.14**
* Source : https://sourceforge.net/projects/magic-stick/files/magic-stick/magic-stick_0.1.14.iso[SourceForge]
* Checksum : `magic-stick_0.1.14.iso.sha256` (même répertoire SourceForge)
* Taille attendue : ~2-4 Go

== Prérequis

* [ ] Clé USB **32 Go minimum** (64 Go recommandé pour A/B + Persistence)
* [ ] PC physique avec port USB bootable
* [ ] ISO téléchargée et checksum vérifié
* [ ] `scripts/flash.sh` disponible (fusionné)
* [ ] Mot de passe sudo connu

== 1. Vérification de l'ISO

=== Téléchargement

[source,bash]
----
# Télécharger ISO + checksum
wget https://sourceforge.net/projects/magic-stick/files/magic-stick/magic-stick_0.1.14.iso/download \
  -O magic-stick_0.1.14.iso
wget https://sourceforge.net/projects/magic-stick/files/magic-stick/magic-stick_0.1.14.iso.sha256/download \
  -O magic-stick_0.1.14.iso.sha256

# Vérifier le checksum
sha256sum -c magic-stick_0.1.14.iso.sha256
----

* [ ] ISO téléchargée
* [ ] Checksum valide
* [ ] Taille > 2 Go
* [ ] `scripts/verify.sh` passe (5/5 checks)

== 2. Flash sur clé USB

[source,bash]
----
cd foundry/public/magic-stick
sudo ./scripts/flash.sh /dev/sdX
----

* [ ] Flash terminé sans erreur
* [ ] `sync` effectué
* [ ] Clé retirable en sécurité
