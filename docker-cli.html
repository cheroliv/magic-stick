Image Docker CLI

L’image cheroliv/magic-stick-cli reproduit la stack logicielle CLI du chroot live-build (sans interface graphique). Cible : jobs CI/CD et Ollama Cloud.

Pull

docker pull cheroliv/magic-stick-cli:latest

Contraintes de securite

  • Aucune cle SSH, aucun token dans l’image

  • Injection au runtime via variables d’environnement ou secrets CI/CD

  • Zero donnee personnelle dans les layers Docker
