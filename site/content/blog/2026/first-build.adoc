.Configuration de live-build
[source,bash]
----
sudo apt install live-build debootstrap squashfs-tools xorriso
lb config -a amd64 -d noble --package-lists xubuntu-desktop
lb build
----

Le premier build est la cle de voute du projet. En utilisant `live-build`, on genere une ISO bootable Xubuntu minimaliste qui servira de base aux partitions A et B.
